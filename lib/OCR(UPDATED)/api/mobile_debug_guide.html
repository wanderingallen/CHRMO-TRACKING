Step 2: XAMPP Configuration Check

Important: Make sure XAMPP is configured to accept connections from other devices.

🔧 Apache Configuration:

1. Open XAMPP Control Panel

2. Click "Config" next to Apache → "httpd.conf"

3. Find the line: Listen 80

4. Make sure it's not configured to listen only on localhost

5. Save and restart Apache

🔥 Windows Firewall:

1. Open Windows Defender Firewall

2. Click "Allow an app or feature through Windows Defender Firewall"

3. Make sure "Apache HTTP Server" is checked for both Private and Public networks

4. If not listed, click "Allow another app..." and add Apache
